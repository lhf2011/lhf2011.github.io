Secondly, for each col who doesn't has 0, find the smallest value; every value in this col minus the smallest value; then, each col will get at least a 0.

Thirdly, cross out the row and the col where 0 located, use as few lines as possible to cover all '0'.

Forthly, do the next step for the values who has not been crossed out.
Find the smallest in all uncrossed values, they all minus the smallest value. For the value who has been crossed 2 times, add this smallest value.
